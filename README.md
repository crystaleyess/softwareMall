# 软件应用商城  

>## 1 项目描述：  

>>该项目的构建采用前后端分离方式，本仓库只包含前端工程，技术栈采用 Vue2.0 + ElementUI + Axios...。目前上传的部分完成了整体的布局，首页以及专题页面，一些组件还待完善，后续将不断更新，敬请期待!
